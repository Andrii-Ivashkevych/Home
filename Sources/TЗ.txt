Техническое задание (рекомендуемое время на реализацию всего ТЗ - до 20 рабочих часов):


HTML:

1. Сверстать psd-шаблон из архива (home.psd)

2. HTML5/CSS3

3. Чистый, семантический, валидный код, кроссбраузерность (Opera, Safari, FireFox, Chrome, IE9+)

4. Основная навигация должна содержать выпадающий список (dropdown menu). Реализовать с помощью css (без использования javascript).

5. Слайдер на главной должен работать, реализация на Ваше усмотрение

6. Функциональные части (tabs, toggles, site tour, ролловеры/ховеры на картинки) должны работать, реализация на Ваше усмотрение

7. Шрифты использовать из Google fonts (http://www.google.com/webfonts)

8. Использовать шрифтовые иконки из любого ресурса например (http://fontello.com/, http://fontastic.me/)


Важно:

1. Как можно меньше картинок и больше CSS

2. Использование CSS3 свойств, а также анимаций

3. Чистота и прозрачность кода


Будет плюсом: 

1. Респонсив от 320px до 1440px

2. Seo разметка и использование микроформатов

3. Правильное использование HTML5 тегов



WordPress:

1. Скачать и установить последнюю версию WordPress

2. Взять за основу последнюю стандартную WordPress тему (twentyfifteen или twentytwelve)

3. Добавить шорткод Latest News с psd-шаблона (стиль вивода должен на 100% соответствовать psd-шаблону)

4. Для реализации использовать:
	- посты блога в качестве данных для вывода шорткода
	- класс WP_Query 
	- стандартные WordPress функции вывода the_date(), the_title()...

5. Шорткод должен иметь следующие параметры:
	- количество выводимых постов
	- Очередность вывода (orderby):
		- date
		- title
		- rand
	- Order:
		- ASC
		- DESC
	- возможность включения/отключения для:
		- thumbnail
		- author
		- comments
		- excerpt
